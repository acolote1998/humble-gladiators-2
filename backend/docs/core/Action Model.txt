Action (@Embeddable)

Integer damageCaused - Damage dealt by this action (null if no damage)
Integer healingCaused - Healing provided by this action (null if no healing)
ActionType actionType - Type of action (HEAL, SPELL, PHYSICATTACK, RANGEDATTACK, ITEM)
StateType stateCaused - State effect applied (NONE, NORMAL, PARALIZED, CONFUSED, FOCUSED, POISONED)

Note: 
- Uses Integer instead of int for nullability
- Embedded in Turn entity
- Lombok @Getter and @Setter annotations
- Fields are private for better encapsulation
